# Zombie Arena
Your objective is to kill all the zombies. Keep an eye at your ammo, as it is limited.
Choose the boost you want before each 
### Commands
- Use your mouse to aim
- Move: WASD
- Shoot: left mouse button
- Recharge: R
- Quit: ESC
